# Examples

- [`3d-world`](3d-world/): flagship Three.js multiplayer example with quantized movement, packet groups, flattened snapshots, volatile updates, reconnect, recovery, and delayed disconnect cleanup.
- [`auth-jwt`](auth-jwt/README.md): authentication and recovery-identity pattern.
- [`rpc`](rpc/README.md): request/respond usage and failure handling.
- [`rooms`](rooms/README.md): server-owned room membership and broadcasts.

The smaller examples are documentation-first so they do not duplicate a complete server scaffold. The 3D world is runnable.
